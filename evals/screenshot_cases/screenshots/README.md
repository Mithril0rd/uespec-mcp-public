# Screenshot Inputs

Place local screenshot-to-UISpec evaluation images in this directory. The source
tree intentionally keeps this folder empty except for documentation/placeholders
because real product captures may be private or non-redistributable.

Expected filenames:
- `combat-hud.png`
- `inventory-panel.png`
- `quest-tracker.png`
- `nameplate-cluster.png`
- `chat-and-social.png`

Use PNG when possible. JPG and WEBP are also accepted by `generate_spec_from_image`.
Do not commit proprietary screenshots.
